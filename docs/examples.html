Examples

Worked notebooks covering single-qubit, driven, multi-qubit, and benchmark scenarios. Each one is a standalone Jupyter notebook and an executable Python script in the repository.

Single-qubit decay

Spontaneous emission of a two-level atom. The smallest non-trivial example — validates LindbladTTN against qutip.mesolve to ~10⁻⁷.

1 qubit PS1 qutip

Driven qubit (Rabi + decay)

A coherent drive $\Omega\cos(\omega_d t)\,\sigma_x$ plus T1 decay. Rabi oscillations damp into the steady state. Demonstrates the f, V single-drive shortcut.

1 qubit time-dep PS1

Bell-state generation

Rx(π/2) on qubit 0 followed by a cross-resonance ZX gate, with realistic T1/T2 decoherence. Tracks concurrence and fidelity to the ideal Bell state.

2 qubits gates T1/T2

Five-qubit cat state

Prepare and propagate a GHZ-style cat state $(|00000\rangle + |11111\rangle)/\sqrt{2}$ with single-qubit decay on each site. Compares train vs tree topology.

5 qubits cat state tree

Cascade with Gaussian drives

Five qubits driven by a cascade of Gaussian pulses on different sites — uses the drives=[(f_i, V_i), …] multi-channel API.

5 qubits multi-drive pulses

Scaling benchmark

Wall-clock time and memory vs system size for $N = 2, 4, 6, 8$ qubits at fixed bond dimension. Includes a CPU↔GPU comparison.

benchmark GPU

Running the examples locally

Each notebook in the gallery corresponds to a script in the examples/ folder. Clone the repository, install LindbladTTN (and the example extras), then run either form:

pip install -e .
pip install matplotlib qutip jupyter

# Run the script directly
python examples/qutip_simple.py

# Or open the notebook
jupyter notebook docs/notebooks/single_qubit_decay.ipynb

The example scripts save figures next to themselves; the notebooks bake the plots directly into their outputs.
