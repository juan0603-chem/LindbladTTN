Citation

How to cite

If LindbladTTN supports work in a publication, please cite the repository. A companion paper is forthcoming.

Software citation

Until the paper is out, cite the GitHub repository directly. A placeholder BibTeX entry is below — update the URL and commit hash to match the version you used.

BibTeX

@software{lindblad_ttn_2026,
  author       = {LindbladTTN contributors},
  title        = {LindbladTTN: Tree Tensor Network Lindblad Master Equation Solver},
  year         = {2026},
  publisher    = {GitHub},
  url          = {https://github.com/<your-org>/lindblad_ttn},
  version      = {0.2.0},
  note         = {Open-source Python library}
}

APA

LindbladTTN contributors. (2026). LindbladTTN: Tree Tensor Network Lindblad
master equation solver (Version 0.2.0) [Computer software].
https://github.com/<your-org>/lindblad_ttn

Background references

If you cite LindbladTTN, please also reference the foundational papers on Tree Tensor Networks and the TDVP, along with the related TENSO project that inspired LindbladTTN's architecture.

  • Schollwöck, U. The density-matrix renormalization group in the age of matrix product states. Annals of Physics 326, 96–192 (2011).
  • Haegeman, J., Lubich, C., Oseledets, I., Vandereycken, B. & Verstraete, F. Unifying time evolution and optimization with matrix product states. Phys. Rev. B 94, 165116 (2016).
  • Chen, X. & Franco, I. Tree tensor network hierarchical equations of motion based on time-dependent variational principle for efficient open quantum dynamics in structured thermal environments. J. Chem. Phys. 163, 104109 (2025). DOI
A note on TENSO

LindbladTTN is architecturally inspired by pytenso but written entirely from scratch and specialised for the Lindblad master equation. If you use both in a study, cite both.
