Getting Started

A complete LindbladTTN workflow in four steps, using single-qubit spontaneous emission — the smallest physically meaningful problem.

The problem: a two-level system with frequency $\omega$ undergoing T1 decay at rate $\gamma$. The state evolves according to the Lindblad master equation:

$$ \frac{d\rho}{dt} = -i\,[H,\rho] + \gamma\left(\sigma_-\,\rho\,\sigma_+ - \tfrac{1}{2}\{\sigma_+\sigma_-,\,\rho\}\right), \qquad H = \tfrac{\omega}{2}\sigma_z. $$

Starting from the excited state $\rho_0 = |1\rangle\langle 1|$, the population $\langle\sigma_z\rangle$ should decay exponentially from $-1$ to $+1$ (ground state).

1. Define the system

Build the Pauli operators, the Hamiltonian $H_0$, and the list of Lindblad jump operators. Each jump operator is a tuple (γ, L): the rate and the operator.

import numpy as np
from lindblad_ttn import LindbladTTN

# Pauli operators
sx = np.array([[0,  1], [1,  0]], dtype=complex)
sy = np.array([[0, -1j], [1j, 0]], dtype=complex)
sz = np.array([[1,  0], [0, -1]], dtype=complex)
sm = np.array([[0,  0], [1,  0]], dtype=complex)   # |0><1| lowering

omega = 1.0   # qubit frequency
gamma = 0.3   # T1 decay rate

H0    = 0.5 * omega * sz
L_ops = [(gamma, sm)]

2. Build the solver

Instantiate LindbladTTN. The four most important arguments are n_sites, bond_dim, topology, and strategy.

solver = LindbladTTN(
    H0=H0,
    f=None, V=None,            # no time-dependent drive
    L_ops=L_ops,
    n_sites=1,                 # one qubit
    bond_dim=4,                # max TTN rank
    topology="train",          # or "tree" for balanced binary tree
    strategy="ps1",            # or "vmf" for adaptive ODE integration
    device="cpu",              # or "cuda"
    dtype="complex128",
)

For a single qubit the bond dimension is over-specified — the state lives in a 4-dimensional Liouville space and even bond_dim=4 is exact. For larger systems, the bond dimension is the key knob between cost and accuracy.

Train or tree?

Use topology="train" for chains and 1D problems, "tree" when every qubit interacts with many others. The two topologies hold the same information; their bond-dimension scaling differs by problem structure.

3. Run the evolution

Call solver.run(...) with an initial density matrix and the time grid. Observables are tracked over time and returned as numpy arrays.

rho0 = np.array([[1.0, 0.0],
                 [0.0, 0.0]], dtype=complex)   # |1><1|, excited state

result = solver.run(
    rho0=rho0,
    t_span=(0.0, 15.0),
    dt=0.01,
    observables=[sz, sx, sy],
    save_every=1,
    verbose=True,
)

The expectation value of an observable $O$ is computed at every saved step as $\langle O\rangle(t) = \mathrm{Tr}\!\left[O\,\rho(t)\right]$.

4. Inspect results

The returned LindbladResult dataclass holds everything the solver saved during the run.

import matplotlib.pyplot as plt

print("Final trace:", result.norm[-1])              # ~1.0 (trace-preserving)
print("Max bond dim:", result.bond_dims.max())

fig, ax = plt.subplots(figsize=(7, 4))
ax.plot(result.times, result.expect[0].real, label=r"$\langle\sigma_z\rangle$")
ax.plot(result.times, result.expect[1].real, label=r"$\langle\sigma_x\rangle$")
ax.plot(result.times, result.expect[2].real, label=r"$\langle\sigma_y\rangle$")
ax.set_xlabel("t"); ax.set_ylabel("expectation"); ax.legend()
plt.show()

# Save a full plain-text report
result.save_txt("single_qubit_decay.txt")

You should see $\langle\sigma_z\rangle$ rise from $-1$ to $+1$ (ground state population) with the time constant $1/\gamma$, while $\langle\sigma_x\rangle$ and $\langle\sigma_y\rangle$ oscillate at $\omega$ and decay at $\gamma/2$. That's the spontaneous emission picture exactly.
