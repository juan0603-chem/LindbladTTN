API Reference

LindbladTTN exports exactly two public names: the solver class and the result dataclass. Everything else is internal.

from lindblad_ttn import LindbladTTN, LindbladResult

class LindbladTTN

The single user-facing class. Constructing it builds the Liouvillian SoP and the TTN topology; calling .run(...) propagates the density matrix.

Constructor arguments

ArgumentTypeDefaultDescription
H0 np.ndarray | None — Static Hamiltonian, shape $(2^N, 2^N)$. May be None for purely dissipative dynamics.
f Callable[[float], float] | None None Drive envelope $f(t)$ for the single-drive shortcut. Pair with V. Mutually exclusive with drives.
V np.ndarray | None None Drive operator for the single-drive shortcut. Combined as $H(t) = H_0 + f(t)\,V$.
drives list[(Callable, ndarray)] | None None Multi-channel drives: each tuple is $(f_i, V_i)$. The resulting Hamiltonian is $H(t) = H_0 + \sum_i f_i(t)\,V_i$.
L_ops list[(float, ndarray)] | None None Lindblad jump operators as (γ, L) tuples. Empty list = unitary evolution.
n_sites int 1 Number of qubits. Must be ≥ 1.
bond_dim int 32 Maximum TTN bond dimension $\chi$.
topology "train" | "tree" "tree" TTN structure. "train" = MPS-style chain; "tree" = balanced binary tree.
strategy "ps1" | "vmf" "ps1" TDVP variant. "ps1" = projector-splitting, "vmf" = variable mean-field ODE.
device str "cpu" PyTorch device string: "cpu", "cuda", "cuda:0", etc.
dtype str | torch.dtype "complex128" Tensor dtype. Use "complex64" on GPU for ~3× speed at ~10⁻⁵ accuracy cost.
num_threads int | None None If set, caps PyTorch's intra-op thread count via torch.set_num_threads(...).
parallel_nodes bool False VMF only. Evaluate per-node RHS terms concurrently inside each ODE step.
vmf_atol float 1e-8 Absolute tolerance passed to torchdiffeq when strategy="vmf".
ode_method str "dopri5" torchdiffeq integrator name. Any torchdiffeq-supported method is accepted.
krylov_dim int 20 Krylov subspace dimension for PS1 local matrix exponentials.
units str "natural" Reserved for future use; currently affects no behaviour.

Method: run(rho0, t_span, dt, observables=None, save_every=1, verbose=True) → LindbladResult

Run the TDVP evolution.

ArgumentTypeDefaultDescription
rho0 np.ndarray — Initial state. Either a state vector (shape $(2^N,)$, projected to $|\psi\rangle\langle\psi|$ automatically) or a density matrix (shape $(2^N, 2^N)$). Normalised internally to unit trace.
t_span tuple[float, float] — Time interval $(t_0, t_f)$.
dt float — Fixed step size for PS1, or initial step for the VMF integrator.
observables list[np.ndarray] | None None List of $(2^N, 2^N)$ observables. Their expectation values are saved at every save_every steps.
save_every int 1 Save the state every $n$ steps. Set higher to reduce memory and I/O.
verbose bool True If true, render a tqdm progress bar during propagation.

Properties

  • solver.frame — the underlying Frame (TTN topology). Read-only.

Example

from lindblad_ttn import LindbladTTN
import numpy as np

sz = np.array([[1, 0], [0, -1]], dtype=complex)
sm = np.array([[0, 0], [1,  0]], dtype=complex)

solver = LindbladTTN(
    H0=0.5 * sz,
    L_ops=[(0.1, sm)],
    n_sites=1,
    bond_dim=4,
    topology="train",
    strategy="ps1",
)
rho0 = np.array([[1.0, 0.0], [0.0, 0.0]], dtype=complex)
result = solver.run(rho0=rho0, t_span=(0.0, 10.0), dt=0.01,
                    observables=[sz])

class LindbladResult

Dataclass returned by solver.run(...).

Attributes

AttributeTypeDescription
times np.ndarray 1D array of saved time points, shape (n_saved,).
expect dict[int, np.ndarray] Maps observable index → complex time series of $\langle O\rangle(t)$, each shape (n_saved,).
rho_final np.ndarray Final density matrix at $t = t_f$, shape $(2^N, 2^N)$.
norm np.ndarray $\mathrm{Tr}\,\rho(t)$ at every saved step — should stay at 1 for a trace-preserving evolution.
bond_dims np.ndarray Max virtual bond dimension over all node-node edges, at every saved step.

Methods

MethodDescription
save_txt(path="lindblad_result.txt") Write a full plain-text report: the final density matrix (real, imag, |·|), summary statistics (trace, purity, Hermiticity error, eigenvalue range), and the full time series with one row per saved step.
