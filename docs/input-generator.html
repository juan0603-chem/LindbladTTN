What this builds

The generator emits a self-contained Python script that defines Pauli operators, embeds them on each qubit, builds the Hamiltonian and a local-decay channel on every site, instantiates LindbladTTN, and runs a single TDVP trajectory from $\rho_0 = |0\ldots 0\rangle\langle 0\ldots 0|$. Edit it further to match your specific physics.
